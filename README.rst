===============
Luce Squishable
===============
--------------------------------------------------
Squishes Luce's lua classes into one portable file
--------------------------------------------------

Usage
=====

   .. code:: bash

       make [LUCE_SOURCES=<path_to_luce_sources>, default: ../luce]

Output
------
    
    a flatly squished ``luce.lua``.
